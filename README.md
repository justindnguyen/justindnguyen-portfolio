## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/justindnguyen/justindnguyen-portfolio.git
   cd justindnguyen-portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build the project:
   ```bash
   npm run build
   ```

5. Deploy to GitHub Pages:
   ```bash
   npm run deploy
   ```
